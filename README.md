# SOC-Homelab-pFsense-Wazuh-Splunk-Cowrie
A full end-to-end SOC home lab environment built for hands-on detection, monitoring, and incident response. Includes pfSense firewall, Wazuh SIEM, Splunk Enterprise, a Windows endpoint, Kali attacker machine, and a Cowrie SSH honeypot. Covers network segmentation, log ingestion, alerting, threat hunting, and blue-team analysis
